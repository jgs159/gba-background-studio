## GBA Background Studio

**GBA Background Studio** एक डेस्कटॉप एप्लिकेशन है जो **Game Boy Advance (GBA) बैकग्राउंड** बनाने और संपादित करने के लिए है। यह छवियों को GBA-संगत tilesets और tilemaps में बदलने, tiles और palettes को विज़ुअली संपादित करने, और आपके GBA प्रोजेक्ट्स के लिए उपयोग के लिए तैयार assets निर्यात करने की सुविधा देता है।

> ⚠️ यह एप्लिकेशन उन डेवलपर्स, ROM hackers और pixel artists के लिए डिज़ाइन की गई है जिन्हें GBA बैकग्राउंड पर सटीक नियंत्रण की आवश्यकता है।

## ✨ विशेषताएं

- **छवि से GBA रूपांतरण**
  - मानक छवियों को GBA-संगत tilesets और tilemaps में बदलता है।
  - आउटपुट आकार और रंग गहराई (4bpp और 8bpp) कॉन्फ़िगर करता है।
  - निर्यात से पहले परिणाम का पूर्वावलोकन।

- **Tileset संपादन**
  - Tiles का विज़ुअल चयन और संपादन।
  - Tilemap ग्रिड पर इंटरैक्टिव ड्राइंग टूल्स।
  - पिक्सेल-दर-पिक्सेल संपादन के लिए 100% से 800% तक ज़ूम स्तर।

- **Palette संपादन**
  - प्रति palette 256 रंगों तक संपादन।
  - Palette परिवर्तनों को पूर्वावलोकन और tiles के साथ सिंक्रनाइज़ करें।
  - व्यक्तिगत रंगों को पुनर्व्यवस्थित, प्रतिस्थापित या समायोजित करें।

- **पूर्वावलोकन टैब**
  - GBA-जैसी स्क्रीन पर आपका अंतिम बैकग्राउंड कैसा दिखेगा, यह देखें।
  - Tile और palette कॉन्फ़िगरेशन को जल्दी से सत्यापित करें।

- **पूर्ववत करें/फिर से करें इतिहास**
  - संपादन इतिहास की पूर्ण ट्रैकिंग।
  - व्यापक इतिहास बफर के साथ पूर्ववत और फिर से करने के ऑपरेशन।

- **कॉन्फ़िगर करने योग्य इंटरफ़ेस और स्टेटस बार**
  - Tile चयन, tilemap निर्देशांक, palette ID, फ्लिप स्थिति और ज़ूम स्तर के साथ विस्तृत स्टेटस बार।
  - प्रति टैब संदर्भ टूलबार (पूर्वावलोकन, tiles, palettes)।

- **बहुभाषी समर्थन**
  - सेटिंग्स के माध्यम से भाषा चयन के साथ आंतरिक अनुवाद प्रणाली (Translator)।
  - इंटरफ़ेस में कई भाषाओं का समर्थन करने के लिए डिज़ाइन किया गया।

## 🏗️ आर्किटेक्चर विवरण

GBA Background Studio **Python** और **PySide6** के साथ बनाया गया है, और एक मॉड्यूलर इंटरफ़ेस डिज़ाइन का पालन करता है:

- **मुख्य विंडो (`GBABackgroundStudio`)**
  - एप्लिकेशन स्थिति प्रबंधित करती है (वर्तमान BPP, ज़ूम स्तर, tile और palette चयन)।
  - मुख्य टैब और कस्टम स्टेटस बार होस्ट करती है।
  - कॉन्फ़िगरेशन लोड और लागू करती है (अंतिम आउटपुट सत्र सहित)।

- **टैब**
  - `PreviewTab` – GBA-शैली बैकग्राउंड पूर्वावलोकन।
  - `EditTilesTab` – Tile और tilemap संपादन टूल्स।
  - `EditPalettesTab` – Palette संपादक और रंग हेरफेर टूल्स।

- **इंटरफ़ेस घटक और उपयोगिताएं**
  - `MenuBar` – फ़ाइल ऑपरेशन (छवि खोलें, फ़ाइलें निर्यात करें, बाहर निकलें) और संपादक क्रियाएं।
  - `CustomGraphicsView` – Tile-आधारित इंटरैक्शन के साथ विस्तारित `QGraphicsView`।
  - `TilemapUtils` – Tilemap इंटरैक्शन और चयन के लिए साझा लॉजिक।
  - `HistoryManager` – संपादक ऑपरेशन के लिए पूर्ववत/फिर से करें प्रबंधन।
  - `HoverManager`, `GridManager` – Hover प्रभाव और ग्रिड ओवरले के लिए विज़ुअल सहायक।
  - `Translator`, `ConfigManager` – स्थानीयकरण और स्थायी कॉन्फ़िगरेशन।

## 📦 इंस्टॉलेशन

## 📦 इंस्टालेशन

### आवश्यकताएँ
- **Python** (3.12+ अनुशंसित)
- **Pip** (पायथन पैकेज मैनेजर)
- **PySide6 के लिए ऑपरेटिंग सिस्टम सपोर्ट:**
  - **Windows:** विंडोज 10 (वर्जन 1809) या बाद का।
  - **macOS:** macOS 11 (Big Sur) या बाद का।
  - **Linux:** glibc 2.28 या बाद के संस्करण के साथ आधुनिक वितरण।

### डिपेंडेंसी
मुख्य डिपेंडेंसी में शामिल हैं:
- `PySide6` (पायथन के लिए Qt) - *नोट: इसके लिए ऊपर बताए गए OS वर्जन की आवश्यकता है।*
- `Pillow` (PIL) इमेज प्रोसेसिंग के लिए।

आप इनका उपयोग करके डिपेंडेंसी इंस्टॉल कर सकते हैं:
```bash
pip install -r requirements.txt
```

---

### 🏛️ पुराने ऑपरेटिंग सिस्टम के लिए सहायता (Windows 7 / 8 / 8.1)
यदि आप विंडोज के पुराने संस्करण का उपयोग कर रहे हैं जो **PySide6** (GUI फ्रेमवर्क) का समर्थन नहीं करता है, तो आप अभी भी हमारे **बहुभाषी कमांड लाइन विज़ार्ड** के माध्यम से मुख्य रूपांतरण इंजन का उपयोग कर सकते हैं।

आप पुराने सिस्टम के लिए डिपेंडेंसी यहाँ से इंस्टॉल कर सकते हैं:
```bash
pip install -r requirements-legacy.txt
```

यह आपको बिना ग्राफिकल इंटरफेस के, अपनी मूल भाषा में स्टेप-बाय-स्टेप विज़ार्ड का उपयोग करके छवियों को GBA एसेट्स में बदलने की अनुमति देता है।

1. प्रोजेक्ट रूट पर जाएँ।
2. **`GBA_Studio_Wizard.bat`** फ़ाइल चलाएँ।
3. अपनी भाषा चुनें (18 भाषाओं का समर्थन)।
4. इमेज ड्रैग-एंड-ड्रॉप करने और GBA आउटपुट कॉन्फ़िगर करने के लिए निर्देशों का पालन करें।

## 🚀 शुरुआत करना

1. **रिपॉजिटरी क्लोन करें**

   ```bash
   git clone https://github.com/CompuMaxx/gba-background-studio.git
   cd gba-background-studio
   ```

2. **वर्चुअल एनवायरनमेंट बनाएं और सक्रिय करें** (वैकल्पिक लेकिन अनुशंसित)

   ```bash
   python -m venv .venv
   source .venv/bin/activate   # Windows पर: .venv\Scripts\activate
   ```

3. **निर्भरताएं इंस्टॉल करें**

   ```bash
   pip install -r requirements.txt
   ```

4. **एप्लिकेशन चलाएं**

   ```bash
   python main.py
   ```

## 🧭 बुनियादी उपयोग

1. **एक छवि खोलें**
   - **फ़ाइल → छवि खोलें** पर जाएं या `Ctrl+O` दबाएं।
   - वह छवि चुनें जिसे आप GBA बैकग्राउंड में बदलना चाहते हैं।

2. **रूपांतरण कॉन्फ़िगर करें**
   - **BG मोड** (**टेक्स्ट मोड** या **रोटेशन/स्केलिंग**) चुनें।
   - उपयोग करने के लिए palette(s) या Tilemap चुनें (केवल **टेक्स्ट मोड 4bpp** के लिए)।
   - वह रंग सेट करें जो पारदर्शी के रूप में उपयोग किया जाएगा।
   - आउटपुट आकार और अन्य आवश्यक पैरामीटर समायोजित करें।
   - **बदलें** पर क्लिक करें और एप्लिकेशन बाकी काम करेगी।

3. **Tiles संपादित करें**
   - **टाइल्स संपादित करें** टैब पर जाएं।
   - व्यक्तिगत tiles को ड्रा और संशोधित करने के लिए tilemap व्यू का उपयोग करें।
   - Tile समूहों को कॉपी, कट, पेस्ट या रोटेट करने के लिए पूरे क्षेत्र चुनें।
   - तत्काल परिणाम देखने के लिए रियल-टाइम में परिवर्तन सिंक्रनाइज़ करें।
   - सटीक संपादन के लिए **ज़ूम** स्तर समायोजित करें।
   - स्थान बचाने या हार्डवेयर संगतता सुनिश्चित करने के लिए टाइल्स ऑप्टिमाइज़/डी-ऑप्टिमाइज़ करें।
   - **4bpp** और **8bpp** फॉर्मेट के बीच assets कनवर्ट करें।
   - **टेक्स्ट मोड** और **रोटेशन/स्केलिंग** के बीच आसानी से स्विच करें।

4. **Palettes संपादित करें**
   - **पैलेट संपादित करें** टैब पर जाएं।
   - Palette ग्रिड में रंग संशोधित करें और रंग संपादक से उन्हें समायोजित करें।
   - किसी palette से संबंधित विशिष्ट क्षेत्र या सभी tiles चुनें और उन्हें दूसरे से बदलें या स्वैप करें।

5. **बैकग्राउंड पूर्वावलोकन**
   - वास्तविक GBA पर कैसा दिखेगा इसके सटीक प्रतिनिधित्व के लिए **पूर्वावलोकन** टैब पर जाएं।
   - सत्यापित करें कि आपकी tile और palette कॉन्फ़िगरेशन एक साथ सही काम करती हैं।

6. **Assets निर्यात करें**
   - **फ़ाइल → फ़ाइलें निर्यात करें** पर जाएं या `Ctrl+E` दबाएं।
   - Tilesets, tilemaps और palettes को आपकी GBA विकास टूलचेन में एकीकरण के लिए तैयार फॉर्मेट में निर्यात करें।
   - यदि आवश्यक हो तो अपने संबंधित मेनू से अलग-अलग assets निर्यात करें।

## 🔄 पूर्ववत करें/फिर से करें

एप्लिकेशन एक **इतिहास प्रबंधक** का उपयोग करके आपकी संपादन क्रियाओं को ट्रैक करती है:

- **पूर्ववत करें (Undo)** – अंतिम ऑपरेशन को वापस करता है।
- **फिर से करें (Redo)** – एक पूर्ववत ऑपरेशन को फिर से लागू करता है।

इतिहास प्रणाली हाल के राज्यों का एक बफर बनाए रखती है, जिसमें tile संपादन, palette परिवर्तन और tilemap ऑपरेशन शामिल हैं।

## ⚙️ कॉन्फ़िगरेशन और स्थानीयकरण

### कॉन्फ़िगरेशन

एप्लिकेशन निम्नलिखित जैसी सेटिंग्स संग्रहीत करने के लिए एक कॉन्फ़िगरेशन मैनेजर का उपयोग करती है:

- अंतिम उपयोग की गई भाषा
- अंतिम उपयोग किया गया ज़ूम स्तर
- शुरू होने पर अंतिम आउटपुट लोड करना है या नहीं
- अन्य इंटरफ़ेस और संपादक प्राथमिकताएं

कॉन्फ़िगरेशन शुरू होने पर लोड होती है और इंटरफ़ेस और मेनू पर लागू होती है।

### स्थानीयकरण

एक `Translator` घटक इंटरफ़ेस टेक्स्ट प्रबंधित करता है:

- डिफ़ॉल्ट भाषा सेटिंग्स के माध्यम से कॉन्फ़िगर की जाती है।
- अधिक भाषाओं का समर्थन करने के लिए अनुवाद फ़ाइलें जोड़ी या संपादित की जा सकती हैं।
- इंटरफ़ेस टेक्स्ट (मेनू, डायलॉग, लेबल) अनुवादक के माध्यम से जाते हैं।
